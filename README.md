## How to Run
1. Set your telegram bot token as environment variable `TELEGRAM_BOT_TOKEN`:
```
export TELEGRAM_BOT_TOKEN=<your_telegram_bot_token>
```

2. Add `src` to `PYTHONPATH`:
```
export PYTHONPATH=${PWD}
```

3. Run:
```
python src/run.py
```
